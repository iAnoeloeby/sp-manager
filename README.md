# Start Page Browser

Start Page Browser is a lightweight, customizable Chrome/Edge new tab extension that turns every browser start page into a clean personal dashboard. It combines a focused search bar, live clock, greeting, quick links, and flexible wallpaper controls in a responsive interface built with React, Vite, and Tailwind CSS.

It is designed to stay minimal while still giving you the essentials you want on a daily start page: fast access to search, an easy way to launch favorite sites, and enough visual customization to make the page feel personal without getting noisy.

## Features

- Dark / Light mode
- Accent color
- Search
- Clock
- Quick Links
- Wallpaper
- Export / Import
- Responsive
