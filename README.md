# Start Page Browser

Start Page Browser is a lightweight, customizable Chrome/Edge new tab extension that transforms every browser start page into a clean personal workspace. Built with React, Vite, and Tailwind CSS, it provides a modular interface where users can organize shortcuts, widgets, and built-in tools in the way that works best for them.

It is designed to stay minimal while giving users complete control over their start page. Every region of the workspace can be customized with different types of content, allowing the page to evolve from a simple dashboard into a workspace tailored to each user's daily workflow.

## Features

### Workspace

* Customizable workspace
* Multiple workspace regions
* Flexible content arrangement
* Persistent layout

### Widgets & Tools

* Search
* Clock
* Quick Links
* Weather
* Planner
* Notes
* More coming soon

### Personalization

* Dark / Light mode
* Accent color
* Wallpaper
* Responsive design

### Data

* Import / Export
* Local storage
